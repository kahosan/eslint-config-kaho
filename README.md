# eslint-config-kaho

## Installation

```bash
pnpm add -D eslint prettier eslint-config-kaho
```

## Usage

### Normal Project for Javascript

```json
{
  "extends": "kaho"
}
```

### Typescript

```json
{
  "extends": "kaho/ts"
}
```

### react

```json
{
  "extends": "kaho/react"
}
```

### vue

```json
{
  "extends": "kaho/vue"
}
```
